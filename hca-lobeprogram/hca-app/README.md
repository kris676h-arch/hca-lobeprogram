# HCA Halvmaraton 2026 — Løbeprogram

Personlig løbeapp til HCA Halvmaraton 27. september 2026. Mål: **1:55:00** (5:27 min/km).

## Sådan får du appen på din telefon

### 1. Upload til GitHub
1. Gå til [github.com](https://github.com) og opret et nyt repository kaldet `hca-lobeprogram`
2. Upload alle filerne fra denne mappe

### 2. Aktivér GitHub Pages
1. Gå til **Settings → Pages** i dit repository
2. Under "Source" vælg **"GitHub Actions"**
3. Appen deployer automatisk når du pusher til `main`

Din app: `https://DITBRUGERNAVN.github.io/hca-lobeprogram/`

### 3. Installér på iPhone (Safari)
1. Åbn linket i Safari
2. Tryk på Del-ikonet (firkant med pil op)
3. Vælg "Føj til hjemmeskærm"

## Lokal udvikling
```bash
npm install
npm run dev
```
